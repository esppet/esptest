This game needs a free Firebase project for Google login and the scoreboard. After you paste your config into firebase-config.js and push to GitHub, reload this page.

  1. Create a project at Firebase Console
  2. Enable Google sign-in under Authentication
  3. Create a Firestore database
  4. Copy your web app config into firebase-config.js
  5. Add esppet.github.io under Authentication → Settings → Authorized domains
